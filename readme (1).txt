HOW TO USE: 

1 - download the code
2 - open terminal
3 - install all dependencies
4 - start mongodb
5 - type "nodemon index.js" in terminal
6 - go to browser on localhost:3000 
7 - SIGNUP and LOGIN 


DEPENDENCIES:
